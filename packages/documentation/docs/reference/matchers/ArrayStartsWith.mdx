Matches an Array which starts with the provided array of matchers - any additional elements in the array are ignored.

<Tabs groupId="language">
  <TabItem value="java" label="Java">
```java
import io.contract_testing.contractcase.definitions.matchers.ArrayStartsWith;

new ArrayStartsWith(java.util.List<java.lang.Object> matchers);
```

#### Parameters <a name="Parameters" id="Parameters"></a>
| **Name** | **Type** | **Description** |
| --- | --- | --- |
| matchers | `java.util.List<java.lang.Object>` |  An array of matchers that describes the start of the array.<br/>Additional elements in the actual array are ignored. |
