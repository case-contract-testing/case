## ObjectEachValueMatches

Matches an object where each value matches the provided matcher.

<Tabs groupId="language">
  <TabItem value="java" label="Java">
```java
import io.contract_testing.contractcase.definitions.matchers.ObjectEachValueMatches;

new ObjectEachValueMatches(java.lang.Object matcher);
```

#### Parameters <a name="Parameters" id="Parameters"></a>
| **Name** | **Type** | **Description** |
| --- | --- | --- |
| matcher | `java.lang.Object` | The matcher that all values must pass. |
