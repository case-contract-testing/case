Saves the matcher below it with a unique name that can be used with lookups in tests after this one.

Lookups can be made with [matchers.convenience.ReferenceMatch](../convenience/ReferenceMatch).

<Tabs groupId="language">
  <TabItem value="typescript" label="Typescript">
```typescript
import { matchers } from '@contract-case/case-definition-dsl'

new matchers.convenience.NamedMatch(name: string, child: any)
```

#### Parameters <a name="Parameters" id="Parameters"></a>
| **Name** | **Type** | **Description** |
| --- | --- | --- |
| name | `string` | The name you can use to lookup this matcher later. |
| child | `any` | The content of this named match.<br/>If omitted or undefined, the content will be looked up in a previously named match |
  </TabItem>

new NamedMatch(java.lang.String name, java.lang.Object child);

#### Parameters <a name="Parameters" id="Parameters"></a>

| --- | --- | --- |
| name | `java.lang.String` | The name you can use to lookup this matcher later. |
| child | `java.lang.Object` | The content of this named match.<br/>If omitted or undefined, the content will be looked up in a previously named match |
