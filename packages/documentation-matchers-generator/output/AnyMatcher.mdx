The base class for all Test Equivalence Matchers.

Extend this if you don't
have an example in your Matcher. Otherwise, use `AnyMatcherWithExample`

<Tabs groupId="language">
  <TabItem value="java" label="Java">
```java
import io.contract_testing.contractcase.definitions.matchers.AnyMatcher;

new AnyMatcher(java.lang.String matcherType);
```

#### Parameters <a name="Parameters" id="Parameters"></a>
| **Name** | **Type** | **Description** |
| --- | --- | --- |
| matcherType | `java.lang.String` | The type string for this matcher (see @link https://case.contract-testing.io/docs/reference/plugin-framework/extending-case Extending ContractCase for a description of these strings). |
